# CV/ML Paper SOP Checklist

- Project:
- Task:
- Benchmark or setting:
- Current stage:

## Stage 1: Scope

- [ ] One-sentence problem statement exists
- [ ] Main bottleneck is explicit

## Stage 2: Read for Boundaries

- [ ] Key papers mapped
- [ ] At least one paper-reading note written
- [ ] One concrete next-step question extracted

## Stage 3: Idea

- [ ] Idea card written
- [ ] Validation path is explicit
- [ ] Main risk is explicit

## Stage 4: Baseline and Minimal Experiment

- [ ] Baseline is trusted
- [ ] First controlled experiment is run
- [ ] Experiment log is updated

## Stage 5: Weekly Loop

- [ ] Weekly review written
- [ ] Current bottleneck is classified
- [ ] Next 1-2 experiments are decided

## Stage 6: Story Lock

- [ ] Claim map exists
- [ ] Major claims have evidence
- [ ] Unsupported claims were weakened or removed

## Stage 7: Section Writing

- [ ] Introduction drafted or revised
- [ ] Method drafted or revised
- [ ] Experiments drafted or revised
- [ ] Abstract aligned with evidence

## Stage 8: Pre-Submission Audit

- [ ] Self-review completed
- [ ] Reviewer-risk list written
- [ ] Figures and tables reviewed for clarity

## Stage 9: Rebuttal

- [ ] Reviewer concerns classified
- [ ] Direct answers written
- [ ] New experiments prioritized only if high impact
